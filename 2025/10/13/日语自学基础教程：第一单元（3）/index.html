一、预览内容

这一节课，最重要的内容是理解和存在句式相关的内容。其中的关键动词 あります 和 います，将在未来的其他句式中再来袭击我们（不是）。

此外，从存在句式开始延展，我们也将初步说明助词 に 的用法。它作用一个表达 “抽象点” 的助词，在非常多的句式中都有使用。
并且，在未来学习各种连用文节（也即状语文节）时，它也会时常出现。

以及我们会简单说说主格助词 が 以及并列格助词 と 的用法。

此外，我们还会比较 に 与 で 之间的微妙差别，并且由此来串联起之前讲过的许多语法，并为未来要学的一些固定句式埋下伏笔。

但是，です 的本质是一种类似 “是” 的绝对判定，也就是说，如果拿出 “Aは Bです” 这样的句子，语感上是 “A ≡ B”，也就是 A 与 B 是两个等价、可互换的物体。

2.2 关于 あります 与 です 的更多关联（补充）

但是，其实 に あります 与 です 的缘分并不止于此。

我们在第一节课，就学习过 です 的否定形式为 では ありません，我当时说过 否定补助动词 的 ありません 更多细节按下不表。然而，现在学习者肯定能一眼看出 あります 与 ありません 之间绝对有关系。

否定助动词 ん

其实，ありません 正是 あります 的否定形式，而它实际否定的意思来自于 否定助动词 ん。
而为了接续这么一个用言性助词，动词 あります 必须要变为连用形，也就是 ありませ，随后再添加助动词 ん。

值得注意的是，这里其实也满足了用言性成分后置的原则。
另外，学习者可能注意到了，我第一课（1）说明时使用的说法是 “否定补助动词 ありません”，
但是我们现在知道真正的 “否定助词” 是 ん，不过理解层次的不同不影响我们的使用
这其实揭露了日语一个非常有意思的点：
由于日语的成分与文节同构，实际上我们可以在完全不了解一个文节的构成的情况下，
仅仅通过它若干特别的属性就明白它的用法，乃至于把它当做一个成分使用。
这就是日语黏着语 “语法构成一块砖，哪有需要哪里搬” 的特性。

当时学习者可能就非常疑惑，为什么 です 的否定形式会如此之长，又为什么 です 的 “连用形” 会是 で。

而且学习者也自然学习到了无生命物存在句式的否定形式为 Aは Bに ありません 了。

另外，为什么 で　あります 会是 “是” 这样的意思呢？
其实这与 で 的含义有关，で 的一种含义我们可以粗略地认为类似 “作为”（类似英语 as），
因此 “作为 xxx 而存在”，就是 “是” 的意思，也因此 です 代表的断定的两个名词是可以互换的。
这也算是对 存在 动词 あります 意义的一个 echo 吧。

2.3 表有生命物的存在句式 に います

表示生命物的存在句式与表示无生命物的存在句式没有本质差别，只是将表示无生命物的存在动词 あります 改为了表示有生命物的第二种存在动词 います。

同样地，这里给出 います 的汉字遣以方便记忆：居ます。

学习者肯定能猜出来生命物的存在句式的是 Aは Bに いません，因此不再赘述。

2.4 敬语助动词 ます

学习者一定注意到了，两个存在动词都是以 ます 结尾的。那么两者共有的这个 ます 是什么呢？

其实就是敬语助动词 ます。

在日语中，ます 几乎无处不在，基本所有敬语动词，都会最终变形以接续 ます，想必听过日语的学习者并不会陌生这个与 です 齐名的助动词。所以说，ます 就是敬语世界的基石。
而如果不考虑复杂的敬语，变形为以 ます 结句的句子，被统称为 敬重语（丁寧語）。

通常在与没有明显上下级、前后辈关系的陌生人交谈时，使用敬重语就已经足够了。假如使用过于高级的敬语，反而会使得对方觉得无所适从，乃至于觉得受到了阴阳。

回到语法部分，我们可以自然得到以下两个结论：

  • あり 和 い 是某个动词的连用形。
  • あります 与 います 有完全一致的活用。

至于 あり 和 い 两个动词的活用，我们此处仅作铺垫，所以按下不表。

2.5 主格助词 が

我在第一单元（1）就已经严肃指出过主题助词 は 并非表示主语的助词。而今天，我们终于要学习到真正表示主语的助词，也就是 が 了。

好吧，其实虽然我之前对主格助词遮遮掩掩的，但是实际上它就和英语中的 subject 基本完全一样，没有 は 那样复杂的语感在其中。

  • 日语中通常会省略掉可以通过上下文推知的主语，因此要慎用主格助词，防止让听话人觉得用力过猛。这一点同样可以辅证第一点，正是因为很少直接说主语，因此显式提出主语的时候会让人觉得语感非常强烈

2.6 两种存在句式的格式比较

由于有生命物和无生命物的存在句式其实并没有本质差别，接下来我将使用 あります 作为例子，说明标日中所说两者存在句式的 格式 的微妙差异。

从 “A 里有 B” 讲起

标日上说，“Aに Bが あります” 是 “A 中有 B” 的意思，以及 “Bは Aに あります” 是 “B 在 A 里” 的意思。一个 A B，另外一个 B A，也许有些初学者会被误导，以为这是日语语序与中文语序相同的证据。为了方便区别，我都使用 A 当做地点，B 当做存在物
然而学习了文节和日语的谓语后置规则后，学习者一定知道这是错误的。

并且，根据文节本身意义的不变性，即使调换文节的位置，“Bが Aに あります” 也应该是 “A 里有 B” 的意思

吗？

很遗憾，其实并非如此简单。

首先，和汉语一样，在日语中也同样有一个语感规则，那就是一句话中，越靠前的文节听起来越重要。
这种感觉就类似于，开头的那个文节自带了 “主题” 或 “强调” 的感觉（即使没有 は）。

比如，“书，放在了这里” 和 “这里，放了书” 虽然构成是相同的，但是语感是不尽相同的。
在日语中同样有这样的规律。因此主题助词 は 所领衔的 主题 文节，常常都在整个句子的最开头，以示强调。

因此，“Aに Bが あります” 和 “Bが Aに あります” 由于开头文节不同，听上去的语感并不相同。

另外，我们上文提到过，が 会导致被提示的那个主语有一种被强调的感觉。
可是如果 が 所在的文节不在最开头，其语感就会被相对削弱。

我们将带有 强调 之意的 は 拿过来进行放大一下，体会就会非常明显：

  • Aには Bが あります：A 里面的话，有着 B 哦。
  • Bは Aに あります：B 的话，在 A 里哦。

“哦” 只是我用来提升 “主题” 感的辅助词，并不是两个例句真的这么俏皮。
另外，表强调/主题的 は 与 に 复合时，是接续在 に 后面
而与 が 复合时，は 则会直接掩蔽掉 が
可以认为这是因为 が 所代表的主语意思可以从上下文推断，但是 に 代表的地点意思却不能

再看 “B 在 A 里”

在上一小节，我已经举出了 “Bは Aに あります” 的例子来说明 B A 语序句子的语感。这是否意味着，同为 B A 语序的 “Bは Aに あります” 和 “Bが Aに あります” 的意思就是一样的呢？

怎么可能嘞？

“Bは Aに あります” 相比于 “Bが Aに あります” 肯定多了一个将主语 B 进行主题化/强调的意思啊。
并且，根据我上文所说 が 的特殊语感，我们可以得到下述这个表：

含义
Bは Aに あります B 的话，在 A 里哦
Bが Aに あります 那可是 B（而不是别的），在 A 里

相较之下，“Aに Bが あります” 和 “Aには Bが あります” 的语感差距则不是很大，基本都是：A 里，可是有个 B 哦。

上述例句中，我基本都使用 “哦” 来强调哪一部分是 “听话人所不知道” 的，而使用 “可是” 来表达听话人所不知道的部分

因此，如果学习者发现两个句式的差别有些烧脑时，就可以试着把第一个文节加上 は 来辅助自己进行理解。

三、时间状语

在第二节中，我提到 に 是表达 “它只负责表达 ‘存在点’” 的副词。实际上，这个 “存在点” 完全不局限于地点，而是广义上的抽象的点。

这其中，就包含 “时间点”。

因此，在标日第五课的例句中，有 “森さんは 七時に 起きます” 这种例句，这正是因为 “七点” 是一个抽象的时间点，而不是时间段，从而使用 に 来进行状语/副词化，或者用日语些的说法，连用化。

相较之下，像 “先週” “今日” 这样的时间 段，就属于一维线而非零维点，所以不使用 に 来进行时间 点 限定。
