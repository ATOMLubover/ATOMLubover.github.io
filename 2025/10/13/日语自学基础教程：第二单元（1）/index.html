一、预览内容

从这一单元开始，我们就要开始 正式 学习和 动词 相关的内容了。这是日语语法当中最复杂的成分，但是这并不是一种 “全无规律” 的复杂，恰恰相反，它的复杂度体现在 “规则太多” 上。
不过学习者无须为此感到害怕或者担心，既然是规律，那就是有迹可循的。同时，这些规律将能够串起日语中许许多多其他的知识，当学习者再进一步学习时，蓦然回首，就会发现原来自己早已自我构筑了一套能够串联起日语大部分知识的框架。

二、再谈助动词 masu

2.1 复习 desu 与 masu

在第一单元，我们就已经学习了两个敬语助动词，它们构成了日语敬语的基石：

肯定形式 否定形式
desu desu deha arimasen
masu masu masen

并且，我在第一单元（3）中已经指出，desu 实际上也是基于 masu 演化而来，因此 desu 有着与 masu 几乎完全一致的 活用。

但是直到这一单元前，我们只学习了 desu/masu 最基础的肯定和否定形式，相当于只学习了英语中的现在时。而接下来，我们将学习 desu/masu 的 “过去时”。

2.2 过去/完成助动词 ta

正如表达 “否定” 时，日语使用的是 否定助动词 n（nu），在表达过去这一时态时，日语使用的是 过去/完成助动词 ta。

想必学习者在经过一整个单元的学习后，已经立刻能够想到 ta 的接续是什么了吧？没错，就是 连用形。

回忆 masu 它们是怎么与 n 组合的：

终止形，顾名思义，就是用于 结句 的活用形，它必然出现在 一句话最终的那个文节 中。
但是这个说法并没有指出它真正的语法含义，我们应当把终止形类比为英语的 “原型”，因此终止形又称为字典形。
另外，终止形既然是出现在最终文节中的动词（假设只有一个用言性成分），根据谓语后置原则，显然终止形是 用言性 的。
我们学习过的 “desu” “masu” “n（nu” 都是它们的终止形。

那么聪明的学习者肯定要抢答了，masu 接续 ta 的结果必然应该是 “maseta” 了对吧？

但是很遗憾，我没有打上着重标记，说明这种符合直觉的组合方式是错误的。

那么脑子有点大病的日本人是如何组合的呢？或者说，他们把 masu 的连用形魔改成了什么样呢？（因为显然，连用形 + 用言的结构是很难打破的，只能在形式本身上下功夫了。未然形：请输入文本）

想要将 masu 变成连用形来接续 ta 的想法确实是正确的，但是这里使用的 masu 连用形的 “形式” 和我们学的有些不太一样，此处应该使用的连用是 mashi。

也就是说，实际的组合结果是：mashita。

我们填充一下表格：

现在时肯定形式 现在时否定形式 过去时肯定形式
desu desu deha arimasen deshita
masu masu masen mashita

为什么会这样呢？难道 masu 的连用形不是 mase 吗？

也并非如此，实际上 mase 和 mashi 都是 masu 的连用形。虽然有些奇怪，但是确实如此，对于十分特殊的 masu，它的连用形有两种形式，在与不同助动词搭配时，有不同的形式。不过，大部分情况下都是使用 mashi 作连用形，只有在否定和一些特定情况下会使用 mase，比如 いらっしゃいませ。

2.2 两种连用形之间的联系（补充）

那么这两种之间连用形之间有没有联系呢？硬背实在是很麻烦啊。

当然有，但是这个联系也许要再往后学习才能体会到它的内在逻辑。不过我可以在这里先打个铺垫。

拿出我们都十分熟悉的五十音图，把 masu 终止形、连用形的尾假名表上去：

あ い う え お
さ x し す せ x

学习者应该立刻就发觉了，其实所谓的活用，只会变形动词的最后一个假名。

而且，活用其实就是在 将最后一个假名的 “段” 进行切换。当然，这仅限于 **五段** 活用动词，他们的 “活用” 就是在一行的 “五段” 上跳来跳去。

终止形作为原型，它稳坐五十音图段的最中间，而 masu 的两个连用形恰好就在它的左右拱卫。

2.3 那么否定形式呢

显然，仅仅学习了肯定形式是不够满足学习者的。我们接下来就来看看日语的特色敬语咒语😃。

首先学习者可能会开始纠结，否定助动词 n 是用言性的，过去助动词 ta 也是用言性的，我们也没有学习到他们的连用形，该怎么组合和协调他们呢？
这个问题确实很好，因为我们确实无法在不引入 n 或 ta 的连用形的情况下组合它们。

那么究竟将 n 还是 ta 变成 体言性 的呢？

这一点日本人已经帮我们确定好了，他们决定：将 n 变为 体言性。

好吧，那么我就来介绍一下 n 的连用……连体型。

不是，为什么会是连体形？不是要变成体言性，应该是选取用言的连用形吗？

2.3.1 关于连体形那点事

话说到这里，我们暂且放下过去时否定形式的问题，先引入一个新的活用形：连体形。

学习者应该养成一个习惯，遇到一个新的成分，先推测它到底是体言性还是用言性。那么连体形到底是体言性还是用言性的呢？

我们不妨先来看看英语中的例子：

显然，一个 形容词 是 非谓语性 的。

如果用日语化的说法呢？把 “形容词”（修饰名词的词）换成 “连体”（连接体言），“非谓语性” 换成 “体言性” 呢？连体形实际上就是 *修饰* 体言的活用形。

也就是说，连体形也是体言性 的，我们甚至可以认为，连体形就是体言。这一点在日语中同样成立。
其实非常好理解，光有连体形和它接续的体言，形成的是 “偏正短语”，它缺少动词这样的谓语性文节，因而无法单独成句。

在弄清楚连体形的词性后，我们自然要 关注它的具体形式。

不过目前来说，我们实际上只学习了 masu 的活用（存在动词 ari 和 i 暂且只学习了连用形），以及助动词 n 和 ta。我们这里暂时只引出 n 的连体形。因为没记错的话，标日的连体形相关内容都被排到初级下册了。

让我们隆重介绍，n 的连体形：n！

……没错，n 的连体形就是 n。

也许是出于和我国推行简体字类似的目的，现代日语中，任何动词的连体形都和终止形一模一样。唯一区分他们的方法，只是观察他们是否是全句的最后一个用言性成分。也许正是因为这其实非常好分辨，所以日语才将这两个活用形的形式归并到了一起。

当然，这样推导可能会有先射箭后画靶的嫌疑。
而然在语言当中，留存的规则不一定是唯一可行的，我们只能证明某个思路是正确的，但是不能证明只有这个思路是正确的。
我认为只要一个理论可以合理解释某个语言现象，那么它就是可取的。
另外，这里推导过程可能学习者不能很好地把握，这是可以理解的。我当初实际上是与上文以相反的思路推导出的结论：
我先从 desu 前应该接续体言的结论出发，进而推测 n 是某种体言性活用形，然后排除诸如终止形等形式的可能后，断定其为连体形。
尤其是在 n 这样一个活用形不完整的助动词上，排除终止形和未然形后，唯一存在的形式就是连体形，从而印证了我的推测。
